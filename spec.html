K-Collusion Index 스펙 정리

한국을 기준값 100으로 두고 주요 G20 회원국의 상대 물가 수준을 비교하는 정적 대시보드 프로젝트의 현재 구현 기준 스펙입니다.

개요 범위 데이터 계산식 화면 구조 빌드/배포 이슈

1. 제품 개요

제품명 K-Collusion Index
핵심 목적 한국 기준 상대 물가 수준을 World Bank WDI country-level 데이터가 있는 G20 회원국별로 비교
기준값 대한민국 KOR = 100
기준 연도 현재 데이터 파일 기준 2024

2. 기능 범위

구분 스펙 현재 상태
대시보드 상대 물가 지수 차트, 국가 순위표, 요약 지표 제공 구현
데이터 로딩 /data/k-collusion-index.json 정적 JSON 파일 사용 구현
다운로드 브라우저에서 CSV 생성 다운로드, JSON 파일 직접 다운로드 구현
World Bank WDI 연동 PPP 기반 price level ratio 수집, World Bank API 불가 시 WDI 동기화 공개 API 사용 구현
정기 업데이트 GitHub Actions 주간 데이터 재생성 및 커밋 구현

3. 데이터 스펙

대상 국가

대상 코드는 ARG, AUS, BRA, CAN, CHN, FRA, DEU, IND, IDN, ITA, JPN, KOR, MEX, RUS, SAU, ZAF, TUR, GBR, USA, EU27입니다.

정적 JSON 구조

{
  "success": true,
  "data": [
    {
      "countryCode": "USA",
      "countryName": "미국",
      "indexValue": 125.5,
      "baseYear": 2024
    }
  ],
  "timestamp": "2026-05-10T08:11:12.236069",
  "baseYear": 2024
}

프론트엔드 타입

필드 타입 설명
countryCode G20CountryCode 국가 코드
countryName string 표시용 국가명
indexValue number 한국 기준 상대 지수
baseYear number 데이터 기준 연도

4. 계산 스펙

지수는 기준 연도의 한국 값을 분모로 두고, 각 국가 값을 상대화해서 산출합니다. 한국은 표시 안정성을 위해 항상 100으로 보정합니다. 

indexValue = round((countryValue / koreaValue) * 100, 2)
KOR.indexValue = 100.0
  • 단일 연도 계산 함수: calculate_k_collusion_index
  • 다중 연도 계산 함수: calculate_multi_year_index
  • 정렬 기준: indexValue 내림차순
  • 현재 정적 데이터 생성 스크립트는 World Bank WDI PA.NUS.PPPC.RF를 우선 사용하고, API가 불가하면 World Bank WDI 동기화 공개 API를 사용합니다.

7. 빌드, 테스트, 배포

로컬 실행

npm install
npm run dev

정적 빌드

npm run build

prebuild 단계에서 python/generate_data.py가 실행되고, Next.js는 out 디렉터리로 정적 export합니다.

CI/CD

  • prebuild.yml: 데이터 생성, Next.js 빌드, 정적 산출물 검증
  • cloudflare-pages.yml: Cloudflare Pages 배포
  • weekly-data-update.yml: 매주 일요일 06:00 UTC 데이터 재생성 및 커밋
CPI update: World Bank PA.NUS.PPPC.RF is retained as the broad cross-country price-level index because the latest complete observed G20 price-level year is 2024. World Bank WDI FP.CPI.TOTL.ZG latest observed annual consumer price inflation is added as supplementary context, not as a replacement ranking metric.

8. 확인된 이슈와 개선 필요사항

CPI 상승률은 국가 간 물가 수준 비교에 부적합하므로 PPP 기반 price level ratio로 교체했습니다. 남은 작업은 EU 집계 처리 정책 결정입니다.
  • World Bank API가 응답하지 않으면 WDI 동기화 공개 API를 사용하고, 모든 온라인 경로가 실패할 때만 2024년 WDI 스냅샷을 사용하며 isFallback으로 표시합니다.
  • EU 집계는 같은 WDI country-level 지표로 일관되게 비교하기 어려워 현재 비교 대상에서 제외했습니다.
  • Cloudflare Pages 대시보드에 build command를 설정하면 out 커밋 의존성을 줄일 수 있습니다.
